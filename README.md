# React Introduction Assignment

Welcome to your first React project! 

### 🚀 How to Deploy
When you are ready to see your site live, click the button below:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

### 📝 Your Task
1. Open `src/App.jsx`.
2. Change the `<h1>` text to your name.
3. Change the background color in `src/App.css`.
4. Push your changes to GitHub, then click the Deploy button above!
